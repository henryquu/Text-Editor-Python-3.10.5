# Text Editor 

Text editor made in Python 3.10.5, using Tkinter for interface

# Shortcuts:
| Option | Shortcut |
|--------|----------| 
| New file | Ctrl + n |
| New window | Ctrl + Shift + n |
| Open file | Ctrl + o |
| Save as | Ctrl + s |
| Find | Ctrl + f |
| Hgihlight all + replace | F2 with a marked pattern |
